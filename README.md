# Web Designing Workshop (Semester 2)

Welcome to the **Web Designing Workshop** repository for Semester 2. This repository contains a structured collection of practical experiments, core concepts, and a final project focused on **JavaScript**, **Bootstrap**, and **React**.

## 🚀 Live Demo
The project and practicals are deployed and can be viewed live here:
👉 **[Live Deployment on Netlify](https://marvelous-caramel-7f057a.netlify.app)**

---

## 📂 Repository Structure

The repository is organized into distinct folders for each experiment and practical session:

### 🔬 Practical Experiments
* **EXP 1 to 6:** Foundational web design exercises covering HTML5, CSS3 layouts, and responsive design principles.
* **Exp 7 to Exp8 / 9:** Core JavaScript integration, DOM manipulation, and event handling.
* **Experiment-10 to 12:** Advanced concepts, introductory React components, state management, and dynamic rendering.

### 💼 Projects & Testing
* **`project sem 2/`** – The main web design project incorporating all semester-long learnings.
* **`Practicalst1/`** – Internal practical assessment files.
* **`sample.html` & `samplebootstrap.html`** – Quick reference boilerplates and Bootstrap framework tests.

---

## 🛠️ Tech Stack Used

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Frameworks & Libraries:** React.js, Bootstrap
* **Environment / Package Management:** Node.js (`package-lock.json`)
* **Hosting:** Netlify
